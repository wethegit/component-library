# icon

### Install

```bash
npx @wethegit/components-cli add icon
```

### Usage

Add `IconDefs` to your app, make sure it's only added once, and place your `IconSymbol` components as children of `IconDefs`.

`IconSymbol` takes an `id` prop, which is used to reference the icon later, and a `size` prop, which is used to scale the icon.

You should use icons with a 1:1 aspect ratio, so they can be scaled relatively and without distortion.

```tsx
import { IconDefs, IconSymbol } from "@local/components"

function RootLayout() {
  return (
    <html lang="en">
      <body>
        <IconDefs>
          <IconSymbol id="play" size={27}>
            <path d="M27 13.2396L5.6875 26.4792L5.6875 -9.31519e-07L27 13.2396Z" />
          </IconSymbol>
        </IconDefs>
        {children}
      </body>
    </html>
  )
}
```

Then you can use the `Icon` component anywhere in your app.

```tsx
import { Icon } from "@local/components"

function Page() {
  return (
    <main>
      <button>
        <Icon id="play" />
        Play video
      </button>
    </main>
  )
}
```
