# classnames

The `classnames` function is a utility for conditionally joining class names together. It takes a variable number of class names as input and returns a string of space-separated class names.

## Parameters

`classes: ClassnamesProps`

- The `classes` parameter represents the class names to be joined. It can be a string, an array of strings, or a nested structure of class names.

## Return Value

The function returns a string containing the concatenated and space-separated class names.

## Example Usage

```typescript
import { classnames } from 'path/to/classnames';

const buttonClasses = classnames('button', 'primary', ['is-active', null, undefined], 1);
```

The above example will return the following:

```typescript
'button primary is-active'
```

## Notes

If the classes parameter is not provided or is `null` or `undefined`, the function returns an empty string.

If a class name is an array or a nested structure of class names, the function flattens the structure and filters out non-string values before joining them.
