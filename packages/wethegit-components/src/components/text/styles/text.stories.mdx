# Text Styles

This file contains the CSS variables that are used to customize the text styles and must be added to your global stylesheet for the `<Text>` component to work properly.

- [Font size](#font-size)
- [Font weight](#font-weight)
- [Line height](#line-height)

### Font size

The `<Text>` component is intended to work with pixel-based font-sizes, as most design software follows this same convention. Thus, all of the CSS custom properties related to the variant prop must take in a _unitless_ pixel value. This value is then converted it to a `rem` value at build time.

For example, these settings…

```css
:root {
  --wtc-font-size-title-1: 42; /* 42px */
  --wtc-font-size-title-2: 33; /* 33px */
  --wtc-font-size-title-3: 26; /* 26px */
  --wtc-font-size-title-4: 20; /* 20px */
  --wtc-font-size-title-5: 16; /* 16px */
  --wtc-font-size-title-6: 13; /* 13px */
  --wtc-font-size-body: 16; /* 16px */
  --wtc-font-size-body-smaller: 13; /* 13px */
  --wtc-font-size-body-larger: 20; /* 20px */
  --wtc-font-size-body-legal: 10; /* 10px */
}
```

…compile to these values:

```css
.variant-title-1 {
  font-size: 2.625rem;
}

.variant-title-2 {
  font-size: 2.0625rem;
}

.variant-title-3 {
  font-size: 1.625rem;
}

.variant-body {
  font-size: 1rem;
}

/* and so on */
```

This conversion is made possible thanks to the Sass helper, [calc-rem](/docs/components-grid-layout-styles-grid-layout-utilities--overview), which is included as part of the `<Text>` component. The function receives a unitless pixel value, and converts it into a `rem` value.

### Font weight

Font weight is controlled by the `weight` prop, and is also managed by CSS custom properties which can be found in the `text.scss` file. The available weights are as follows, and can be adjust however you need, based on your font files.

```css
:root {
  --wtc-font-weight-light: 300;
  --wtc-font-weight-regular: 400;
  --wtc-font-weight-medium: 500;
  --wtc-font-weight-semibold: 600;
  --wtc-font-weight-bold: 700;
  --wtc-font-weight-black: 900;
}
```

### Line height

Line height is inherited based on the `variant` prop. By default, `body-` variants receive a general "body" line height, and `title-` variants receive a general "heading" line height. These are managed by CSS custom properties which can be found in the `text.scss` file.

```css
:root {
  --wtc-line-height-heading: 1.2;
  --wtc-line-height-body: 1.4;
}
```
