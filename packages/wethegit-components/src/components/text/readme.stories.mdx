# Text

`<Text>` provides a componentized solution to typography styling in a project. It gives us a predictible API to work with for styling purposes, and it allows us to separate _semantic_ hierarchy from _visual_ hierarchy in our typography.

- [Install](#install)
- [Usage](#usage)

  - [Font weight](#font-weight)

## Install

```bash
npx @wethegit/components-cli add text
```

Then add the required typography styles to your global stylesheet:

```css filename="global.scss"
@import "<path-to>/components/text/styles/text.scss";
```

## Usage

The `<Text>` component uses root-level CSS custom properties (i.e. `--property-name`) to handle concerns such as size, weight, and line-height. All of these settings are exposed via the `text.scss` stylesheet for easy adjustment.

### Font size

Font size is controlled by the `variant` prop on the `<Text>` component. Available variants are: `title-1`, `title-2`, `title-3`, `title-4`, `title-5`, `title-6`, `body`, `body-larger`, `body-smaller`, `body-legal`.

```html
import { Text } from "@local/components"

<Text variant="title-1" as="h3">This is a heading</Text>
```

The `<Text>` component is intended to work with pixel-based font-sizes, as most design software follows this same convention. Thus, all of the CSS custom properties related to the variant prop must take in a _unitless_ pixel value. This value is then converted it to a `rem` value at build time.

For example, these settings‚Ä¶

```css
:root {
  --wtc-font-size-title-1: 42; /* 42px */
  --wtc-font-size-title-2: 33; /* 33px */
  --wtc-font-size-title-3: 26; /* 26px */
  --wtc-font-size-title-4: 20; /* 20px */
  --wtc-font-size-title-5: 16; /* 16px */
  --wtc-font-size-title-6: 13; /* 13px */
  --wtc-font-size-body: 16; /* 16px */
  --wtc-font-size-body-smaller: 13; /* 13px */
  --wtc-font-size-body-larger: 20; /* 20px */
  --wtc-font-size-body-legal: 10; /* 10px */
}
```

‚Ä¶compile to these values:

```css
.variant-title-1 {
  font-size: 2.625rem;
}

.variant-title-2 {
  font-size: 2.0625rem;
}

.variant-title-3 {
  font-size: 1.625rem;
}

.variant-body {
  font-size: 1rem;
}

/* and so on */
```

This conversion is made possible thanks to the Sass helper, [calc-rem](#calc-rem), which is included as part of the `<Text>` component. The function receives a unitless pixel value, and converts it into a `rem` value.

### Font weight

Font weight is controlled by the `weight` prop, and is also managed by CSS custom properties which can be found in the `text.scss` file. The available weights are as follows, and can be adjust however you need, based on your font files.

```css
:root {
  --wtc-font-weight-light: 300;
  --wtc-font-weight-regular: 400;
  --wtc-font-weight-medium: 500;
  --wtc-font-weight-semibold: 600;
  --wtc-font-weight-bold: 700;
  --wtc-font-weight-black: 900;
}
```

```html
import { Text } from "@local/components"

<Text variant="body-larger" weight="bold">This is some bold text üòè</Text>
```

### Line height

Line height is inherited based on the `variant` prop. By default, `body-` variants receive a general "body" line height, and `title-` variants receive a general "heading" line height. These are managed by CSS custom properties which can be found in the `text.scss` file.

```css
:root {
  --wtc-line-height-heading: 1.2;
  --wtc-line-height-body: 1.4;
}
```

## Helpers

The `<Text>` component comes with some helpful Sass helper functions, which you can use throughout your codebase.

### calc-rem

Signature: `calc-rem(unitless-pixel-value, <base-unitless-size> = 16)`
`calc-rem` Receives a unitless pixel value, and converts it into a `rem` value. This is accomplished by dividing the value by 16, which is the default font size in most browsers.

#### Example

```css
@use "@local/components/text/styles/text-helpers" as *;

.myTitle {
  font-size: calc-rem(24); // evaluates to 1.5rem
}

.blueBorder {
  border: calc-rem(8) solid blue; // 0.5rem border-width
}
```

### calc-rem-css-var

Accomplishes the same thing as [calc-rem](#calc-rem) but uses a CSS custom property key as the input value. For example:

#### Example

```css
@use "@local/components/text/styles/text-helpers" as *;

:root {
  --my-custom-font-size: 60;
}

.myCustomTitle {
  /* evaluates to 3.75rem */
  font-size: calc-rem-css-var(--my-custom-font-size);
}
```
