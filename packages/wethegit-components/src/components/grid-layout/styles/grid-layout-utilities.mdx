# Grid utilities

The grid layout comes with some Sass utilities to help you size elements based on the grid's configuration.

- [get-column-width](#get-column-width)

## Usage

`@use` the helpers within in a `.scss` stylesheet:

```css filename="components/your-component/your-component.module.scss"
@use "@local/components/grid-layout/styles/grid-layout-utilities" as *;
```

## get-column-width

Signature: `get-column-width(number-of-columns)`

`get-column-width` Receives a number, and returns a CSS calc expression that evaluates to the number of columns multiplied by the grid's column width.

### Example

```css
.customComponent {
  width: get-column-width(6);
}
```
