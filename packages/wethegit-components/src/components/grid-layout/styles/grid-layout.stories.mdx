# Grid Layout Styles

This file contains the CSS variables that are used to calculate the grid layout and it must be imported in your global stylesheet for the grid to work.

You can override these to customize the grid.

## CSS Variables

### `--wtc-total-width`

The total width of the grid layout.

### `--wtc-number-of-columns`

The number of columns in the grid layout.

### `--wtc-gutter-width`

The width of the gutter between columns.

### `--wtc-row-width`

The width of a row in the grid layout. Automatically calculated based on the other variables.
