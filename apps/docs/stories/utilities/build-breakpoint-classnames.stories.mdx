# buildBreakpointClassnames

The `buildBreakpointClassnames` function is designed to generate class names based on breakpoints for responsive design. It takes a [breakpoint-object](/docs/core-breakpoints--docs) property, a styles object, and a style name as input and returns an array of class names.

## Parameters

#### `prop: T | Partial<Omit<Breakpoints<T>, "sm">> | undefined`

- The `prop` parameter represents the breakpoint value or an object containing specific breakpoints (md, lg, xl, xxl).
- It accepts a generic type `T` which can be `string`, `number`, or `boolean`.
- If `prop` is an object, it can have properties for medium (md), large (lg), extra-large (xl), and double extra-large (xxl) breakpoints.

#### `styles: Record<string, string>`

- The `styles` parameter is an object that maps breakpoint values to corresponding class names.

The function will look for the following format in the styles object:

1. `md`: `${styleName}-${value}`
2. `lg`: `${styleName}-lg-${value}`
3. `xl`: `${styleName}-xl-${value}`
4. `xxl`: `${styleName}-xxl-${value}`

#### `styleName: string`

- The `styleName` parameter represents the base name of the style for which the breakpoint class names are generated.

## Return Value

The function returns an array of class names [ClassnamesProps](/docs/utilities-classnames--docs#parameters), representing the generated breakpoint class names.

## Example Usage

```typescript
import { buildBreakpointClassnames } from 'path/to/buildBreakpointClassnames';
import styles from 'path/to/styles.module.css';

const breakpointClasses = buildBreakpointClassnames({md: 2, lg: 3, xl: 4, xxl: 5 }, styles, 'container');
```

The above example will return the following array:

```typescript
[
  'container-2',
  'container-lg-3',
  'container-xl-4',
  'container-xxl-5'
]
```
