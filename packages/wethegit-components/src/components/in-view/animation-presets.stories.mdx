# Animation presets

The animation presets allow you create reusable CSS transition effects for your `<InView>` components.

- [Animation definitions](#animation-definitions)
- [Options](#options)

## Animation definitions

All animations are set up as a Sass variable — `$animation` — which you can customize as needed.

⚠️ If using TypeScript, Please ensure that any animations added to this list are also added to the avaialable list of animation keys within the `./in-view/components/in-view-item/animation/animation.ts` file.

Signature:

```css
$animation: ("key": (values));
```

The `$animation` variable is a SASS map of `animation-preset` keys to `transform` and `opacity` values. All `<InView>` animations read from special `start-` and `end-` values (see example below). Each animation definition will generate a **CSS module** with the className corresponding to the `key` defined here. These will be available on the `animation` object within your JSX files.

### Example

Creating a `scaleUp` animation preset. This will start at `opacity: 0` and `scale(0.5)`, and end at `opacity: 1` and `scale(1)`.

```css
$animation: (
  "scaleUp": (
    "start-alpha": 0,
    "start-rotate": 0deg,
    "start-scale": 0.5,
    "start-x": 0,
    "start-y": 0,
    "end-alpha": 1,
    "end-rotate": 0deg,
    "end-scale": 1,
    "end-x": 0,
    "end-y": 0
  ),
  /* etc. */
);
```

Using the `scaleUp` preset with the `<InViewItem>` component:

```jsx
import { InView, InViewItem, animation } from "@local/components/in-view"

function Component() {
  return (
    <InView>
      <InViewItem animation={animation.scaleUp}>Watch me scale!</InViewItem>
    </InView>
  )
}
```

## Options

The full list of animation `start-` and `end-` values is below.

### Opacity

`start-alpha`, `end-alpha`

### Rotation

`start-rotate`, `end-rotate`

### Scale

`start-scale`, `end-scale`

### Translate X

`start-x`, `end-x`

### Translate Y

`start-y`, `end-y`

### Timing

`duration`, `delay`, `ease`
