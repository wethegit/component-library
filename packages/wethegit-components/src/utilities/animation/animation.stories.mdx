# Animation

The `animation` utility exposes an object that contains a collection of CSS modules which map to animation preset styles.

Animations are set up as a collection of CSS custom properties. They are organized into `--start` and `--end` properties, and correspond to the initial transform/opacity values, and the final ones. It is assumed that all **end** values are browser "defaults", so `--end-x` is `0`, for example. You can edit the corresponsing `--end` values too though, if needed.

```css
--start-alpha: 1;
--start-rotate: 0deg;
--start-scale: 1;
--start-x: 0;
--start-y: 0;
```

To change or add animation presets, update the `utilities/animation/styles/_animation-utilities.scss` file. For more info, check out the guide to [creating custom animations](/docs/utilities-animation-styles-animation-utilities--overview).

## Usage

Using the `scaleUp` preset with the `<InViewItem>` component:

```jsx
import { InView, InViewItem } from "@local/components/in-view"
import { animation } from "@local/utilities/animation"

function Component() {
  return (
    <InView>
      <InViewItem animation={animation.scaleUp}>Watch me scale!</InViewItem>
    </InView>
  )
}
```
